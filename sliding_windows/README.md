# Sliding windows: Practice problems

1. Maximum sum of a fixed-size window.
2. Moving averages of length `k`.
3. Minimum-length positive-integer subarray with sum at least target.
4. Longest substring without repeated characters.
5. Longest substring with at most `k` distinct characters.
6. Minimum window substring containing all characters of `t`, including multiplicity.
7. Count positive-integer subarrays with product below `k`.
