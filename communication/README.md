# Communication drills: Practice problems

For every technical answer, practice this order:

1. Headline
2. Two-to-four-part framework
3. Details or code
4. Tradeoffs and assumptions
5. Recommendation

Exercises:

1. Give a two-minute project summary covering problem, individual contribution, technical decision, impact, and lesson.
2. You are told only, “Engagement is down.” Ask clarifying questions and build a diagnostic tree.
3. Five stakeholders request analyses but capacity exists for two. Explain prioritization and communication.
