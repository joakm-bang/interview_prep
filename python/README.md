# Python fundamentals and binary search: Practice problems

1. **Frequency table:** Count hashable values without `Counter`; handle empty input and state complexity.
2. **First non-repeating character:** Return its index or `-1`.
3. **Group anagrams:** Group a list of strings by anagram class.
4. **Top-k frequent values:** Solve by sorting and by heap; compare complexity.
5. **Longest consecutive sequence:** Expected `O(n)` time on an unsorted integer array.
6. **Robust numeric mean:** Accept `None`, numeric strings, malformed strings, NaN, and infinities.
7. **Exact binary search:** Given a sorted list and target, return the target index or `-1`.
8. **Leftmost insertion point:** Return the first index at which a target could be inserted without breaking sorted order.
9. **Binary search on the answer:** Given positive package sizes and a target number of packages, find the minimum capacity that can process everything within the target number of batches.
