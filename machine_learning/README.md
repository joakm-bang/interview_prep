# Regression and machine learning: Practice problems

1. Separate OLS assumptions for unbiasedness, conventional standard errors, and causal interpretation.
2. Interpret logistic coefficients, odds ratios, marginal effects, and average marginal effects.
3. Compare class weighting, upsampling, downsampling, and threshold selection for rare binary outcomes.
4. Choose a classifier threshold when false negatives cost 20 times false positives.
5. List at least five train/test leakage paths.
6. Design a time-based validation split.
7. Explain calibration curves, Brier score, Platt scaling, and isotonic regression.
8. Diagnose low training error and high validation error.
9. Investigate offline-to-online degradation: covariate shift, label shift, concept drift, logging, and feedback loops.
