# Intervals, scheduling, and heaps: Practice problems

1. **Merge intervals:** Implement both conventions for touching intervals.
2. **Meeting rooms:** Minimum concurrent rooms.
3. **Insert interval:** Insert into sorted non-overlapping intervals and merge.
4. **Running median:** `add` in `O(log n)`, `median` in `O(1)` using two heaps.
5. **Gas-station circuit:** Return a valid start or `-1`; explain the greedy reset.
